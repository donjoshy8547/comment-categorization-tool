In your terminal or command prompt, navigate to the project folder and run:

‚úÖ Step 2: Download NLTK Resources (One-Time Setup)
Run the following in a Python shell or script:

nltk.download('omw-1.4')

This enables text preprocessing functions like stopword removal and lemmatization.
